# Lessons

Pitfalls hit and the fixes that worked. Append new entries at the top so the most recent is always first.

When `/end-session` asks "did we hit any sneaky bugs?", new entries land here. When `/slim-agent-md` extracts pitfall-shaped content from agent instruction files, this is its first-class destination.

---

## Template for new entries

```markdown
## YYYY-MM-DD — Short title

**Symptoms**: What you saw. Error messages, broken behavior, missing output.

**Cause**: Root cause. Often surprising. Cite file:line if relevant.

**Fix**: What made it work. Prefer the minimal change.

**Frequency**: one-time | recurring | always

**Discovered**: YYYY-MM-DD
```
